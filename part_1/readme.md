# Part 1

This part show what a neural network computes.

It covers:

- Input
- Parameter
- Weight
- Bias
- Neuron
- Weighted sum
- Activation function
- Prediction

The implementation contains one artificial neuron. No automatic differentiation, tensors or machine-learning libraries.
